# Cybersecurity Website

A responsive website focused on cybersecurity awareness and basic online safety practices.

## 🌐 Overview

This project was developed as part of my learning process in web development, with the goal of presenting cybersecurity concepts in a simple and accessible way.

## 🎯 Objectives

- Practice front-end development (HTML, CSS, JavaScript)
- Build a responsive layout
- Improve UI structure and organization
- Apply basic interactivity (mobile navigation menu)

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript

## 📱 Features

- Responsive design (mobile-friendly)
- Hamburger menu navigation
- Structured content sections
- Simple and clean UI
